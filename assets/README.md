# Replacing photos & video with your own

All images and video in this folder are **free stock placeholders** used so the
site looks and feels finished while real photos are collected. Every page also
carries a small disclaimer near its photo gallery saying these are
representative stock images, not real photos of the school or children — please
keep that disclaimer visible until real photos are in place, and remove it once
you've swapped everything below.

To replace an image or video with your own, **keep the exact same filename**
and just overwrite the file — the website will pick it up automatically with
no code changes needed. If you want to use a different filename, update the
matching `src`/`url(...)` reference in `index.html` or `styles.css`.
